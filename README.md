# Finance & Banking Data Analytics – Personal Finance Project

This repository contains a personal finance analytics project using Python and SQLite.

## Project 1 – Personal Finance Analytics

**Goal:**  
Analyze bank-like transaction data and generate monthly income vs expense summary and visualization.

### Files

- `personal_finance_analytics/Notebooks/personal_finance.ipynb`  
  Contains:
  - Data loading
  - SQLite in-memory database
  - SQL query for monthly income and expense
  - Matplotlib graph
  - Export to `monthly_summary.csv`

- `personal_finance_analytics/Notebooks/transactions.csv`  
  Sample transaction data.

- `personal_finance_analytics/Notebooks/monthly_summary.csv`  
  Output summary generated.
